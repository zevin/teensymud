Class TelnetFilter
In: lib/network/protocol/telnetfilter.rb
Parent: Filter

The TelnetFilter class implements the Telnet protocol.

This implements most of basic Telnet as per RFCs 854/855/1129/1143 and options in RFCs 857/858/1073/1091
